# AI Password Strength & Crack Predictor

A cybersecurity project using Machine Learning to analyze password strength and estimate password crack difficulty.

## Features
- AI-based password strength prediction
- Entropy calculation
- Crack difficulty estimation
- Password improvement suggestions
- Streamlit web interface

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- Machine Learning
- Cybersecurity Concepts
